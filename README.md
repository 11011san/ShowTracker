# Show Tracker

Is a Android app for tracking your shows what episodes you have seen and when is the next one coming out.


## Features

- Add shows
- Mark eps as seen
- View episodes by:
  - Show
  - Day
  - Week
  - Month
- Offset dated by +/- 7
- Summery information with banners
